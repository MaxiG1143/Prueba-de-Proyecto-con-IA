## 🛍️ Página E-commerce generada con IA

Se incorporó al proyecto una página de tienda online completamente funcional, diseñada y generada con inteligencia artificial como parte del flujo de desarrollo.

---

### 🤖 Herramienta utilizada

**Claude Haiku 4.5** — modelo de lenguaje de Anthropic, utilizado para generar la estructura HTML, los estilos CSS y la lógica de la interfaz de la tienda.

---

### ⚙️ Proceso de desarrollo

1. **Definición del objetivo** — Se describió a la IA el tipo de tienda, los componentes necesarios (catálogo de productos, tarjetas, botones de compra, etc.) y el estilo visual deseado.
2. **Generación del código** — Claude Haiku 4.5 produjo el código completo de la página, incluyendo estructura, diseño y contenido de ejemplo.
3. **Integración al proyecto** — El código generado fue revisado, adaptado e integrado al repositorio desde Visual Studio Code.
4. **Ajustes finales** — Se realizaron correcciones manuales para alinear la página con el resto del proyecto.
